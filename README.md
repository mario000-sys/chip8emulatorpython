# chip8emulatorpython
chip8emulatorpython very early in development

to use do the command pip install -r requirements.txt
then just run it and then choose the rom you wanna play


CHIP-8 Keypad Layout (Mapped to QWERTY):

Original     Keyboard
----------------------
1 2 3 C   =>  1 2 3 4
4 5 6 D   =>  Q W E R
7 8 9 E   =>  A S D F
A 0 B F   =>  Z X C V
